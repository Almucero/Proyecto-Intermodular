# Angular Docs Confluence

## Flujo
1. Analizar componente o servicio objetivo.
2. Buscar pagina existente en Confluence.
3. Actualizar si existe o crear si no existe.
4. Incluir referencia a ticket Jira asociado.

## Plantillas
- Componente: descripcion, inputs, outputs, dependencias, rutas.
- Servicio: metodos, endpoints, dependencias, errores.
