# Sprint Dashboard

## Flujo
1. Consultar sprint activo con JQL.
2. Agrupar issues por estado y prioridad.
3. Calcular metricas de avance.
4. Generar reporte en markdown con formato de sprint-report.
5. Publicar en Confluence.
