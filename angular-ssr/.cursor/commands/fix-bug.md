# Correccion de bug

1. Leer y resumir el bug reportado.
2. Localizar archivos y flujo afectados.
3. Identificar causa raiz concreta.
4. Aplicar la correccion minima necesaria.
5. Añadir o ajustar test de regresion cuando aplique.
6. Ejecutar validaciones relevantes.
7. Entregar resultado con:
   - causa raiz,
   - archivos modificados,
   - estado de validacion.
