# Security Policy

## Supported Versions

Dado el alcance de este proyecto, solo la versión más reciente se tomará en cuenta para posibles revisiones. Las versiones anteriores no reciben ningún tipo de mantenimiento.

| Version | Supported          |
| ------- | ------------------ |
| 1.x.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

Si encuentras alguna vulnerabilidad de seguridad o un fallo importante al revisar o probar este código, te agradecería que lo reportaras de manera privada.

Para no exponer el fallo de forma pública, **por favor no abras un *issue***. En su lugar, envía un correo electrónico detallando el problema a: `alvarokilor@gmail.com`.

Ten en cuenta que este es un proyecto de carácter académico, por lo que no existe un equipo de soporte ni tiempos de respuesta garantizados. Dependiendo de la gravedad del problema, la situación y el estado actual del proyecto, el fallo reportado podrá ser parcheado, modificado, o simplemente tenido en cuenta como referencia para el futuro.
